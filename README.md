# 🧠 Laboratório Azure Speech Studio & Language Studio

Este repositório documenta a prática e os aprendizados adquiridos durante o laboratório focado nas ferramentas **Azure Speech Studio** e **Language Studio**. O objetivo é desenvolver soluções com base em inteligência artificial voltadas para **voz e linguagem natural**, utilizando os serviços da Microsoft Azure.

---

## 🎯 Objetivo do Laboratório

- Praticar o uso de ferramentas de IA da Azure aplicadas à análise de fala e linguagem
- Criar soluções inteligentes que utilizem reconhecimento de fala e compreensão de linguagem natural
- Documentar o processo e extrair insights aplicáveis a projetos reais

---

## 🛠️ Ferramentas Utilizadas

- [Azure Speech Studio](https://speech.microsoft.com/)
- [Azure Language Studio](https://language.azure.com/)
- Azure Portal
- Serviços Cognitivos (Cognitive Services)
- Python (para integração via SDK REST opcional)

---

## 📚 Conteúdo Prático

1. Configuração de Recursos no Azure
2. Uso do Speech Studio: Transcrição e Texto para Fala
3. Uso do Language Studio: Extração de Entidades e Intenção
4. Análise de Resultados e Testes com Áudios Reais
5. Aplicações Reais em Atendimento Automatizado, Chatbots e Acessibilidade
